Guia inteligente para começar

Use estes blocos como base. Comece pelo HTML, conecte o CSS, ligue o JavaScript e evolua por partes.

1. Estrutura

Crie a página com cabeçalho, conteúdo principal e rodapé. Use classes e IDs com nomes claros.

2. Visual

Defina cores, fontes, espaçamento e responsividade antes de adicionar muitos detalhes.

3. Ação

No JavaScript, selecione elementos, escute eventos e atualize a tela com funções pequenas.

script.js 
const actionButton = document.querySelector("#actionButton");
const themeButton = document.querySelector("#themeButton");

function showMessage() {
  alert("Seu JavaScript está funcionando!");
}

function toggleTheme() {
  document.body.classList.toggle("light");
}

actionButton.addEventListener("click", showMessage);
themeButton.addEventListener("click", toggleTheme);

Receita rápida para criar qualquer código

  1. Escreva o objetivo em uma frase: "quero uma calculadora", "quero uma galeria", "quero um menu".
  2. No HTML, crie os elementos que precisam aparecer na tela.
  3. No CSS, organize layout primeiro: largura, grid, flex, espaçamento e cores.
  4. No JS, pegue os elementos com querySelector e crie eventos com addEventListener.
  5. Teste uma parte pequena, corrija, depois adicione a próxima função.
